# General Theory of Finite Obstruction Calculus

Open preprint archive for the General Theory of Finite Obstruction Calculus.

A system should not guess whether a state, transition, proof, repair, query, program, or model update is valid. It should compute the obstruction, certify repair when possible, and otherwise return a typed refusal, lift boundary, or bounded unknown.

This is not a theory of consciousness, semantics, physics, society, or intelligence. It is a finite mathematical framework for obstruction, repair, descent, lift, refusal, and observer-relative certification.

## Core Object

Given a finite cochain complex:

```text
C^0 --d0--> C^1 --d1--> C^2

with d1 ∘ d0 = 0

exact repair lives in

B^1 = im(d0)

and irreducible obstruction lives in

Q^1 = C^1 / B^1

For a declared finite observer/gauge,

Phi([omega]) = inf over alpha in C^0 of || omega - d0(alpha) ||

measures quotient obstruction. Closure failure is measured by

Gamma(omega) = || d1(omega) ||

When Phi_0, the guarded closure ratio is

R_cl = Gamma/Phi.

A local decrease in Phi is not valid if it worsens closure, hidden assumptions, rollback compatibility, or representation discipline.

## Claim Discipline

Every imported idea must do one of four things:

1. prove;
2. descend to finite obstruction-calculus terms;
3. become explicitly conditional/speculative;
4. or be rejected.

No scalar loss, narrative plausibility, model confidence, or local improvement replaces the evaluator law.
